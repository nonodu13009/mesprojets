Raccourcis clavier Mac

Leçon 1 · Chapitre 6 · Les touches essentielles pour coder sur MacBook Pro

Pourquoi ce chapitre ?

Avant d'aller plus loin dans le JavaScript, un petit détour pratique. Sur un clavier Mac AZERTY, les caractères qu'on utilise constamment en code — accolades { }, crochets [ ], backticks ` — ne sont pas visibles sur les touches. Il faut connaître les combinaisons par cœur.

Cette page te servira de référence rapide. Garde-la sous la main le temps de les mémoriser.

Les touches spéciales du Mac

Symbole Nom Position sur le clavier
⌘ Command (Cmd) À côté de la barre d'espace
⌥ Option (Alt) À côté de Command
⇧ Shift (Maj) Flèche vers le haut, au-dessus de Ctrl
⌃ Control (Ctrl) En bas à gauche
fn Function Tout en bas à gauche

Les caractères spéciaux pour coder

C'est LA partie la plus importante. Sur un Mac AZERTY, ces caractères ne sont pas affichés sur les touches — il faut connaître les combinaisons :

Caractère Nom Raccourci Mac Usage en JS
` Backtick ⌥ + \ Template literals
{ Accolade ouvrante ⌥ + ( Objets, blocs if/for
} Accolade fermante ⌥ + ) Fermer un bloc
[ Crochet ouvrant ⌥ + ⇧ + ( Tableaux, index
] Crochet fermant ⌥ + ⇧ + ) Fermer un tableau
| Pipe (barre verticale) ⌥ + ⇧ + L OR logique ||
\ Antislash ⌥ + ⇧ + / Caractères d'échappement
~ Tilde ⌥ + N Chemin home, NOT binaire

Astuce mémo : la touche ⌥ (Option) est la clé pour accéder à presque tous les caractères spéciaux. Retiens surtout : ⌥ + ( pour { et ⌥ + ) pour }.
