Script.js — Premiers pas

Leçon 1 · Chapitre 1 · Lier JavaScript au HTML, console.log, variables et opérateurs

Par où commencer ?

JavaScript est le langage qui rend les pages web interactives. Le HTML structure le contenu, le CSS le met en forme, et JavaScript lui donne vie : réagir aux clics, afficher des données, communiquer avec un serveur...

Mais avant de faire tout ça, il faut savoir comment écrire du JavaScript et où le mettre. C'est le sujet de ce premier chapitre.

Les 3 façons d'ajouter du JavaScript dans une page HTML

Il existe trois méthodes pour intégrer du JavaScript dans un document HTML. Chacune a ses usages et ses limites.

Méthode Où ? Quand l'utiliser ?
1. En ligne (inline) Directement dans une balise HTML Presque jamais — difficile à maintenir
2. Interne (balise script) Dans une balise <script> dans le HTML Pour des tests rapides ou des pages uniques
3. Externe (fichier .js) Dans un fichier .js séparé La méthode recommandée pour tous les projets

1. JavaScript en ligne (inline)

Le code est écrit directement dans un attribut HTML :

<!-- À éviter : mélange HTML et JS -->
<button onclick="alert('Bonjour !')">Cliquez</button>

2. JavaScript interne (balise script)

Le code est écrit dans une balise <script> à l'intérieur du fichier HTML :

<script>
  console.log("Je suis dans le HTML");
</script>

3. JavaScript externe (fichier .js)

Le code est dans un fichier séparé, relié avec l'attribut src :

<script src="script.js"></script>

Bonne pratique : toujours utiliser un fichier externe. Cela sépare le contenu (HTML) du comportement (JS), ce qui rend le code plus lisible, réutilisable et facile à déboguer.

Où placer la balise <script> ?

On la place juste avant </body> pour que le HTML soit entièrement chargé avant l'exécution du script : 

<body>
  <!-- tout le contenu HTML ... -->

  <script src="script.js"></script>
</body>

console.log — ton premier outil

console.log() est la toute première commande que tu vas utiliser. Elle affiche un message dans la console du navigateur (pas sur la page web). C'est ton outil de débogage n°1.

consoleobjet .accès logméthode ( "Bonjour" )appel + argument 
console.log("Bonjour, je suis un script JavaScript.");

Autres méthodes utiles : console.error(), console.warn(), console.table(). Même principe : objet . méthode ( ).

Les variables — stocker des valeurs

Une variable est une boîte avec une étiquette. Tu y mets une valeur (un nombre, du texte, etc.) et tu la retrouves plus tard en utilisant son nom. En JavaScript, on déclare une variable avec let ou const : 

let nombre = 10;       // modifiable
let nombre2 = 20;      // modifiable
const PI = 3.14159;    // non modifiable (constante)
Mot-clé Modifiable ? Quand l'utiliser
const Non Par défaut — pour tout ce qui ne change pas
let Oui Quand la valeur doit évoluer (compteur, score...)
var Oui Jamais — ancien mot-clé, source de bugs

Les opérateurs arithmétiques

Maintenant que tu sais stocker des nombres, tu peux les calculer. JavaScript fournit tous les opérateurs mathématiques classiques :

Opérateur Calcul Résultat
+ Addition 10 + 20 30
- Soustraction 10 − 20 -10
* Multiplication 10 × 20 200
/ Division 10 ÷ 20 0.5
% Modulo Reste de 10 ÷ 20 10
** Puissance 1020 100000000000000000000 
let nombre = 10;
let nombre2 = 20;

console.log(nombre + nombre2);  // 30
console.log(nombre - nombre2);  // -10
console.log(nombre * nombre2);  // 200
console.log(nombre / nombre2);  // 0.5
console.log(nombre % nombre2);  // 10
console.log(nombre ** nombre2); // 100000000000000000000

Le modulo (%) retourne le reste d'une division entière. 10 ÷ 20 = 0 avec un reste de 10. C'est très utile pour savoir si un nombre est pair (n % 2 === 0).

Les raccourcis d'assignation

Au lieu d'écrire nombre = nombre + 10, tu peux utiliser un raccourci : 

let nombre = 100;

nombre += 10;  // nombre = nombre + 10  → 110
nombre -= 10;  // nombre = nombre - 10  → 100
nombre *= 10;  // nombre = nombre * 10  → 1000
nombre /= 10;  // nombre = nombre / 10  → 100

À toi de jouer

  1. Ouvre index.html dans le navigateur
  2. Appuie sur F12 (ou clic droit → Inspecter)
  3. Va dans l'onglet Console — tu verras les résultats du script
  4. Tape let x = 5; puis x * 3 et observe le résultat
  5. Teste le modulo : 10 % 3 — quel est le reste ?

La suite

Tu sais maintenant écrire du JavaScript, afficher des résultats dans la console et faire des calculs avec des variables. Mais console.log n'affiche rien sur la page — l'utilisateur ne voit rien.
