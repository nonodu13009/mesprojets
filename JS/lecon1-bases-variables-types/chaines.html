Les chaînes de caractères

Leçon 1 · Chapitre 4 · Créer, manipuler et combiner du texte

Le type le plus utilisé

Tu sais maintenant que string est l'un des types primitifs de JavaScript. En pratique, c'est celui que tu manipuleras le plus souvent : noms d'utilisateurs, emails, messages, URLs, contenu de formulaires... tout est du texte.

Ce chapitre te montre comment créer des chaînes, les assembler, accéder à leurs caractères et utiliser les méthodes intégrées qui les rendent si pratiques.

Trois façons de créer une chaîne

"Bonjour"guillemets doubles 'Bonjour'guillemets simples `Bonjour`backticks (template) 
const prénom = "Jean-Michel";
const ville = 'Marseille';
const phrase = `Bonjour tout le monde`;

console.log(prénom); // "Jean-Michel"
console.log(ville);  // "Marseille"
console.log(phrase); // "Bonjour tout le monde"

Les guillemets doubles et simples sont équivalents. Les backticks (`) offrent des fonctionnalités supplémentaires qu'on va voir juste après.

Concaténation — assembler des chaînes

On peut coller des chaînes entre elles avec l'opérateur + : 

const prénom = "Jean-Michel";
const message = "Bonjour " + prénom + " !";

console.log(message); // "Bonjour Jean-Michel !"

Attention aux espaces : "Bonjour" + "Jean" donne "BonjourJean". Il faut ajouter un espace dans la chaîne.

Template literals — la méthode moderne

Avec les backticks (`), on peut insérer des variables directement dans la chaîne grâce à ${...}. C'est plus lisible que la concaténation : 

const prénom = "Jean-Michel";
const age = 30;

// Concaténation classique (difficile à lire)
const msg1 = "Je m'appelle " + prénom + " et j'ai " + age + " ans.";

// Template literal (plus clair)
const msg2 = `Je m'appelle ${prénom} et j'ai ${age} ans.`;

console.log(msg2); // "Je m'appelle Jean-Michel et j'ai 30 ans."

On peut aussi mettre des calculs dans ${...} :

console.log(`Dans 10 ans, j'aurai ${age + 10} ans.`);
// "Dans 10 ans, j'aurai 40 ans."

Retiens : dès que tu mélanges du texte et des variables, utilise les backticks `...${variable}...`. C'est la méthode recommandée en JavaScript moderne (ES6+).

Longueur et accès aux caractères

La propriété .length donne le nombre de caractères. Chaque caractère a un index qui commence à 0 :

Bindex 0 oindex 1 nindex 2 jindex 3 oindex 4 uindex 5 rindex 6 
const mot = "Bonjour";

console.log(mot.length);           // 7
console.log(mot[0]);               // "B" (premier caractère)
console.log(mot[3]);               // "j"
console.log(mot[mot.length - 1]);  // "r" (dernier caractère)

Attention : l'index commence à 0, pas à 1. Le premier caractère est à l'index 0, le deuxième à l'index 1, etc.

Méthodes utiles sur les chaînes

Les chaînes ont de nombreuses méthodes intégrées. Voici les plus courantes — tu les utiliseras constamment :

Méthode Rôle Exemple Résultat
.toUpperCase() Tout en majuscules "bonjour".toUpperCase() "BONJOUR"
.toLowerCase() Tout en minuscules "BONJOUR".toLowerCase() "bonjour"
.trim() Supprime les espaces aux extrémités " hello ".trim() "hello"
.includes() Vérifie si un texte est présent "Bonjour".includes("jour") true
.indexOf() Position de la première occurrence "Bonjour".indexOf("j") 3
.slice() Extrait une partie "Bonjour".slice(0, 3) "Bon"
.replace() Remplace du texte "Bonjour".replace("jour", "soir") "Bonsoir"
.split() Découpe en tableau "a,b,c".split(",") ["a", "b", "c"]

Un exemple concret — nettoyer un email saisi par un utilisateur : 

const email = "  Jean@Gmail.COM  ";

// Nettoyer l'email
const clean = email.trim().toLowerCase();
console.log(clean); // "jean@gmail.com"

// Vérifier si c'est un Gmail
console.log(clean.includes("gmail")); // true

// Extraire le nom avant le @
const nom = clean.slice(0, clean.indexOf("@"));
console.log(nom); // "jean"

Important : les méthodes de chaînes ne modifient jamais la chaîne d'origine (les strings sont immuables). Elles retournent toujours une nouvelle chaîne.

Caractères spéciaux

Certains caractères ne peuvent pas être tapés directement dans une chaîne. On utilise un antislash (\) pour les insérer :

Séquence Caractère
\n Retour à la ligne
\t Tabulation
\\ Antislash
\" Guillemet dans une chaîne 
console.log("Ligne 1\nLigne 2");
// Affiche :
// Ligne 1
// Ligne 2

// Avec les backticks, le retour à la ligne est naturel :
console.log(`Ligne 1
Ligne 2`);
// Même résultat, plus lisible

À toi de jouer

  1. Ouvre la console du navigateur (F12 → Console)
  2. Tape "Bonjour".toUpperCase() et appuie sur Entrée
  3. Essaie les autres méthodes : "hello world".includes("world")
  4. Teste les template literals : `J'ai ${10 + 5} ans`
  5. Nettoie un email : " Test@Gmail.COM ".trim().toLowerCase()
