Leçon 1

Les bases — Variables, types et premiers pas en JavaScript

Bienvenue dans ta première leçon

Tu débutes en JavaScript ? Parfait, tu es au bon endroit. Cette leçon te donne les fondations : comment écrire du code JavaScript, le lier à une page HTML, manipuler des variables, comprendre les types de données et travailler avec du texte.

Pas besoin de connaissances préalables en JavaScript — juste un navigateur web (Chrome ou Firefox) et un éditeur de code (VS Code). On va tout construire ensemble, étape par étape.

Comment travailler ?

Chaque page contient des explications, des exemples de code et une section "À toi de jouer" avec des exercices à tester dans la console du navigateur. Pour ouvrir la console :

  1. Ouvre n'importe quelle page dans Chrome ou Firefox
  2. Appuie sur F12 (ou ⌘ + ⌥ + J sur Mac)
  3. Clique sur l'onglet Console
  4. Tu peux taper du JavaScript directement ici et appuyer sur Entrée

Quand tu auras terminé

À la fin de cette leçon, tu sauras déclarer des variables, comprendre les différents types de données, manipuler du texte et comparer des valeurs. Tu seras prêt pour la Leçon 2 : boucles, conditions et fonctions.
