Leçon 4

Manipulation du DOM

Où en es-tu ?

Depuis la leçon 1, tout ton code s'exécute dans la console : tu calcules, tu affiches des messages, tu manipules des objets... mais la page web, elle, ne bouge pas.

Cette leçon change tout : tu vas apprendre à utiliser JavaScript pour lire et modifier la page elle-même — changer un texte, créer des éléments, réagir aux clics, modifier les styles. C'est le pont entre ton code et ce que voit l'utilisateur.

L'outil qui rend ça possible s'appelle le DOM (Document Object Model) : la représentation de la page HTML sous forme d'objets JavaScript.

Ce que tu vas apprendre

Cette leçon est organisée en 3 chapitres progressifs, avec des exemples concrets à tester directement dans la console, sur n'importe quelle page.

Comment aborder cette leçon ?

Suis les chapitres dans l'ordre : d'abord comprendre ce qu'est le DOM et comment il représente la page, ensuite apprendre à sélectionner et modifier des éléments, et enfin agir sur leur apparence avec les styles et les classes CSS.

Le gros avantage de cette leçon : tu peux expérimenter sur n'importe quel site web. Ouvre la console sur cette page même et amuse-toi à la modifier — c'est le meilleur terrain de jeu.

Prérequis

Avant de commencer, assure-toi d'avoir terminé la Leçon 3 : les fonctions fléchées, les objets et leurs méthodes sont utilisés à chaque page. Le DOM est justement un gigantesque objet avec des méthodes — tout ce que tu viens d'apprendre va prendre vie.
