Obtenir et Manipuler des Éléments

Leçon 4 · Chapitre 31 · Les techniques essentielles du DOM

Le plan du chapitre

C'est le chapitre central de la leçon. Trois compétences à acquérir, toujours dans le même ordre logique :

  1. Sélectionner un élément de la page
  2. Lire ou modifier son contenu et ses attributs
  3. Créer de nouveaux éléments et les insérer

1. Sélectionner : querySelector

La méthode moderne pour attraper un élément est document.querySelector(). Elle accepte n'importe quel sélecteur CSS — exactement les mêmes que dans tes feuilles de style : 

// Par balise : le PREMIER h2 de la page
const titre = document.querySelector("h2");

// Par classe (avec le point)
const carte = document.querySelector(".lesson-card");

// Par id (avec le dièse)
const menu = document.querySelector("#menu");

// Sélecteurs combinés, comme en CSS
const lien = document.querySelector("footer a");

Important : querySelector renvoie le premier élément trouvé, ou null si aucun ne correspond. Vérifie toujours dans la console avant de manipuler.

Sélectionner plusieurs éléments : querySelectorAll

Pour attraper tous les éléments qui correspondent, utilise querySelectorAll. Elle renvoie une NodeList, qu'on parcourt avec forEach : 

const paragraphes = document.querySelectorAll("p");

console.log(paragraphes.length); // le nombre de <p> de la page

// Fléchée + forEach : la leçon 3 en action
paragraphes.forEach(p => {
    console.log(p.textContent);
});

Tu croiseras aussi les anciennes méthodes getElementById("menu") et getElementsByClassName("carte") dans du code existant. Elles fonctionnent, mais querySelector / querySelectorAll les remplacent avantageusement : une seule syntaxe pour tous les cas.

2. Lire et modifier le contenu

Une fois l'élément sélectionné, deux propriétés principales pour son contenu : 

const titre = document.querySelector("h1");

// textContent : le texte brut (sûr et rapide)
console.log(titre.textContent);
titre.textContent = "Nouveau titre";

// innerHTML : le contenu avec les balises HTML
const section = document.querySelector("section");
console.log(section.innerHTML);
section.innerHTML = "<strong>Texte en gras</strong>";

Sécurité : n'utilise jamais innerHTML avec du contenu saisi par un utilisateur (formulaire, URL...) : un visiteur malveillant pourrait injecter du code (attaque XSS). Pour du texte, textContent est toujours le bon choix.

Lire et modifier les attributs

Les attributs HTML (href, src, alt...) sont accessibles directement ou via getAttribute / setAttribute : 

const lien = document.querySelector("a");

// Lecture
console.log(lien.href);
console.log(lien.getAttribute("href"));

// Modification
lien.href = "https://developer.mozilla.org";
lien.setAttribute("target", "_blank");

3. Créer et insérer des éléments

Le DOM permet aussi de fabriquer de nouveaux éléments de zéro. Le processus est toujours le même : créer, remplir, insérer. 

// 1. Créer l'élément
const nouveauParagraphe = document.createElement("p");

// 2. Le remplir
nouveauParagraphe.textContent = "Je suis né en JavaScript !";

// 3. L'insérer dans la page
document.body.appendChild(nouveauParagraphe); // à la fin du body

Pour insérer ailleurs qu'à la fin, prepend (au début) et before / after (autour d'un élément) complètent la panoplie. Et pour supprimer : 

// Supprimer un élément de la page
nouveauParagraphe.remove();

Exemple complet : générer une liste

Combinons tout avec un array d'objets (chapitre 28) pour générer du HTML à partir de données — le cœur de toute application web : 

const taches = [
    { titre: "Réviser la leçon 3", faite: true },
    { titre: "Tester le DOM", faite: false },
    { titre: "Créer un mini-projet", faite: false }
];

// On crée une liste <ul>
const liste = document.createElement("ul");

// Un <li> par tâche
taches.forEach(({ titre, faite }) => {
    const item = document.createElement("li");
    item.textContent = faite ? `✔ ${titre}` : titre;
    liste.appendChild(item);
});

// On insère la liste dans la page
document.body.appendChild(liste);

Relis ce code : fléchées, destructuration, ternaire, template literals, forEach... toute la leçon 3 est là, au service du DOM.

À toi de jouer

  1. Ouvre la console (F12 → Console) sur cette page
  2. Sélectionne le h1 et change son textContent
  3. Compte les paragraphes avec document.querySelectorAll("p").length
  4. Change le href du premier lien de la page
  5. Crée un <p> avec createElement, remplis-le et ajoute-le au body
  6. Reproduis l'exemple de la liste de tâches avec tes propres données
  7. Supprime ta liste avec .remove()
