La boucle for

Leçon 2 · Chapitre 7 · Répéter des actions un nombre connu de fois

Pourquoi les boucles ?

Jusqu'ici, chaque instruction de ton code s'exécutait une seule fois. Mais imagine que tu veuilles afficher les nombres de 1 à 100, ou parcourir une liste de 50 produits. Écrire 100 lignes de console.log() serait absurde.

Les boucles (loops) résolvent ce problème : elles permettent de répéter un bloc de code autant de fois que nécessaire, automatiquement. La boucle for est la première que tu vas apprendre — c'est aussi la plus utilisée.

Anatomie de la boucle for

La boucle for se compose de trois parties, séparées par des points-virgules, qui contrôlent son fonctionnement :

for (initialisation ; condition ; incrémentation { ... }bloc de code répété 
for (let i = 0; i < 5; i++) {
    console.log(i);
}
// Affiche : 0, 1, 2, 3, 4

Décomposons les 3 parties :

Partie Exemple Rôle
Initialisation let i = 0 Crée un compteur, exécutée une seule fois au début
Condition i < 5 Testée avant chaque tour. Si false, la boucle s'arrête
Incrémentation i++ Exécutée après chaque tour (augmente le compteur)

Astuce : i++ est un raccourci pour i = i + 1. Tu l'as vu en leçon 1 avec les opérateurs d'assignation.

Exemple : compter de 1 à 10

for (let i = 1; i <= 10; i++) {
    console.log(`Nombre : ${i}`);
}
// Nombre : 1
// Nombre : 2
// ...
// Nombre : 10

Remarque que cette fois on démarre à 1 et on utilise <= pour inclure 10 dans le résultat.

Compter à l'envers

On peut aussi décrémenter le compteur avec i-- : 

for (let i = 5; i >= 1; i--) {
    console.log(`Compte à rebours : ${i}`);
}
console.log("Décollage !");
// Compte à rebours : 5
// Compte à rebours : 4
// ...
// Décollage !

Parcourir un tableau

L'utilisation la plus fréquente de for est de parcourir les éléments d'un tableau (array). Tu te souviens des index qui commencent à 0 ? C'est exactement le même principe qu'avec les chaînes de caractères : 

const fruits = ["pomme", "banane", "cerise"];

for (let i = 0; i < fruits.length; i++) {
    console.log(fruits[i]);
}
// pomme
// banane
// cerise

Rappel : fruits.length donne le nombre d'éléments du tableau (ici 3). On démarre à 0 et on va jusqu'à < length (pas <=) car les index vont de 0 à 2.

La boucle for...of (syntaxe moderne)

Si tu n'as pas besoin de l'index et que tu veux simplement accéder à chaque élément, for...of est plus lisible : 

const fruits = ["pomme", "banane", "cerise"];

for (const fruit of fruits) {
    console.log(fruit);
}
// pomme
// banane
// cerise

Plus besoin de gérer un compteur i — JavaScript s'en charge. C'est la syntaxe à privilégier quand tu veux juste lire les valeurs.

Exemple concret : calculer une moyenne

const notes = [15, 12, 18, 9, 14];
let somme = 0;

for (const note of notes) {
    somme = somme + note;
}

const moyenne = somme / notes.length;
console.log(`Moyenne : ${moyenne}`);
// Moyenne : 13.6

On utilise une variable somme déclarée avant la boucle (sinon elle serait réinitialisée à chaque tour). Ce pattern « accumulateur » revient constamment en programmation.

À toi de jouer

  1. Ouvre la console (⌘ + ⌥ + J)
  2. Tape for (let i = 0; i < 5; i++) { console.log(i); }
  3. Essaie un compte à rebours avec i--
  4. Crée un tableau et parcours-le avec for...of
  5. Calcule la somme des nombres de 1 à 100 avec une boucle

La suite

La boucle for est parfaite quand tu sais à l'avance combien de tours effectuer. Mais parfois, tu ne le sais pas : tu veux répéter tant qu'une condition est vraie — par exemple, demander un mot de passe jusqu'à ce qu'il soit correct. C'est le rôle de la boucle while.
