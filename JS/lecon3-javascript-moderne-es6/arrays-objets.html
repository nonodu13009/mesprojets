Les Arrays d'Objets

Leçon 3 · Chapitre 28 · La structure de données du web

La structure la plus courante du web

Une liste de produits sur un site e-commerce, les résultats d'une recherche, les contacts de ton téléphone, une réponse d'API... presque toutes les données du web sont des tableaux d'objets : 

const produits = [
    { nom: "Clavier", prix: 49, stock: 12 },
    { nom: "Souris", prix: 29, stock: 0 },
    { nom: "Écran", prix: 199, stock: 5 },
    { nom: "Casque", prix: 89, stock: 3 }
];

Chaque élément du tableau est un objet avec la même structure. Pour accéder à une valeur, on combine index et propriété : 

console.log(produits[0]);      // { nom: "Clavier", prix: 49, stock: 12 }
console.log(produits[0].nom);  // "Clavier"
console.log(produits[2].prix); // 199

Parcourir : for...of et forEach

Deux façons classiques de parcourir la liste. Remarque la fonction fléchée et la destructuration, vues aux chapitres précédents : 

// Avec for...of
for (const produit of produits) {
    console.log(`${produit.nom} : ${produit.prix} €`);
}

// Avec forEach + fléchée + destructuration dans le paramètre
produits.forEach(({ nom, prix }) => {
    console.log(`${nom} : ${prix} €`);
});

find : trouver UN objet

find renvoie le premier élément qui satisfait une condition (ou undefined si aucun) : 

const ecran = produits.find(p => p.nom === "Écran");
console.log(ecran); // { nom: "Écran", prix: 199, stock: 5 }

const introuvable = produits.find(p => p.prix > 500);
console.log(introuvable); // undefined

filter : garder PLUSIEURS objets

filter renvoie un nouveau tableau avec tous les éléments qui passent le test : 

// Tous les produits disponibles
const enStock = produits.filter(p => p.stock > 0);
console.log(enStock.length); // 3 (la souris est exclue)

// Tous les produits à moins de 100 €
const abordables = produits.filter(p => p.prix < 100);
console.log(abordables); // Clavier, Souris, Casque

map : transformer chaque objet

map renvoie un nouveau tableau de la même taille, où chaque élément a été transformé : 

// Extraire juste les noms
const noms = produits.map(p => p.nom);
console.log(noms); // ["Clavier", "Souris", "Écran", "Casque"]

// Créer une version avec prix TTC (rappel : parenthèses autour de l'objet !)
const avecTTC = produits.map(p => ({
    nom: p.nom,
    prixTTC: p.prix * 1.2
}));
console.log(avecTTC[0]); // { nom: "Clavier", prixTTC: 58.8 }

Chaîner les méthodes

La vraie puissance : comme filter et map renvoient des tableaux, tu peux les enchaîner pour exprimer une logique complexe en quelques lignes lisibles : 

// Les noms des produits en stock à moins de 100 €
const resultat = produits
    .filter(p => p.stock > 0)
    .filter(p => p.prix < 100)
    .map(p => p.nom);

console.log(resultat); // ["Clavier", "Casque"]

Bonus : pour additionner des valeurs, il existe reduce — par exemple produits.reduce((total, p) => total + p.prix, 0) donne la somme des prix (366). Retiens surtout find, filter et map : ce sont les trois que tu utiliseras tous les jours.
