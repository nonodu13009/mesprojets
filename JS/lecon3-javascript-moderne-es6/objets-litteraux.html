Les Objets Littéraux

Leçon 3 · Chapitre 25 · La structure de données au cœur de JavaScript

C'est quoi, un objet littéral ?

Un objet littéral est une collection de paires clé / valeur écrite directement entre accolades { }. C'est LA structure de données de JavaScript : un utilisateur, un produit, une réponse d'API... tout est objet. 

const utilisateur = {
    nom: "Nogaro",
    prenom: "Jean-Michel",
    age: 30,
    estActif: true
};

Chaque ligne est une propriété : une clé (nom), deux-points, puis une valeur ("Nogaro"). Les valeurs peuvent être de n'importe quel type : chaîne, nombre, booléen, tableau, fonction... ou même un autre objet.

Lire une propriété : point ou crochets

Deux notations pour accéder à une valeur : 

// Notation par point : la plus courante
console.log(utilisateur.nom);       // "Nogaro"

// Notation par crochets : la clé est une chaîne
console.log(utilisateur["nom"]);    // "Nogaro"

// Les crochets acceptent une variable — impossible avec le point !
const cle = "prenom";
console.log(utilisateur[cle]);      // "Jean-Michel"

Règle simple : utilise le point par défaut. Passe aux crochets uniquement quand la clé est dans une variable ou contient des caractères spéciaux (espaces, tirets).

Modifier, ajouter, supprimer

Un objet déclaré avec const n'est pas figé : c'est la référence qui est constante, pas le contenu. Tu peux donc modifier ses propriétés librement : 

// Modifier une propriété existante
utilisateur.age = 31;

// Ajouter une nouvelle propriété
utilisateur.ville = "Marseille";

// Supprimer une propriété
delete utilisateur.estActif;

console.log(utilisateur);
// { nom: "Nogaro", prenom: "Jean-Michel", age: 31, ville: "Marseille" }

Objets imbriqués

Une valeur peut être un autre objet ou un tableau. C'est comme ça que sont structurées la plupart des données réelles : 

const agence = {
    nom: "Nogaro & Boetti",
    adresse: {
        ville: "Marseille",
        codePostal: "13008"
    },
    activites: ["auto", "habitation", "santé"]
};

// On chaîne les points pour descendre dans la structure
console.log(agence.adresse.ville);   // "Marseille"
console.log(agence.activites[0]);    // "auto"

Les raccourcis ES6

ES6 a introduit des raccourcis d'écriture que tu verras partout dans le code moderne.

1. Propriétés raccourcies

Quand le nom de la variable est identique au nom de la clé, inutile de le répéter : 

const nom = "Nogaro";
const age = 30;

// Avant ES6
const personne = { nom: nom, age: age };

// Depuis ES6 : raccourci
const personne2 = { nom, age };

console.log(personne2); // { nom: "Nogaro", age: 30 }

2. Clés calculées

Tu peux utiliser une expression comme nom de clé avec des crochets : 

const champ = "email";

const formulaire = {
    [champ]: "jm@exemple.fr"
};

console.log(formulaire.email); // "jm@exemple.fr"

Parcourir un objet

Trois outils pour explorer les clés et valeurs d'un objet : 

const scores = { maths: 15, anglais: 12, sport: 18 };

console.log(Object.keys(scores));    // ["maths", "anglais", "sport"]
console.log(Object.values(scores));  // [15, 12, 18]
console.log(Object.entries(scores)); // [["maths", 15], ["anglais", 12], ["sport", 18]]

// Boucle sur les paires clé/valeur
for (const [matiere, note] of Object.entries(scores)) {
    console.log(`${matiere} : ${note}/20`);
}

À toi de jouer

  1. Ouvre la console (F12 → Console)
  2. Crée un objet livre avec un titre, un auteur et un nombre de pages
  3. Affiche le titre avec la notation par point, puis par crochets
  4. Ajoute une propriété annee, puis modifie le nombre de pages
  5. Crée deux variables marque et prix, puis un objet avec le raccourci { marque, prix }
  6. Teste Object.keys(livre) et Object.values(livre)

La suite

Tu sais créer et manipuler des objets. Mais extraire leurs valeurs une par une (const nom = utilisateur.nom...) devient vite répétitif. ES6 a une solution élégante : la déstructuration.
