Leçon 3

JavaScript Moderne — ES6 et Plus

Où en es-tu ?

Dans les leçons précédentes, tu as appris les fondations de JavaScript : les variables, les types, les boucles, les conditions et les fonctions. Avec ces outils, tu es capable d'écrire des programmes qui fonctionnent.

Mais si tu regardes du code JavaScript écrit aujourd'hui — dans un projet React, une application Node.js ou même un tutoriel récent — tu vas tomber sur des syntaxes que tu ne reconnais pas encore : des => partout, des objets manipulés dans tous les sens, des { } à gauche d'un =...

C'est parce que JavaScript a énormément évolué depuis 2015 avec la mise à jour ES6 (ECMAScript 2015). Cette leçon va te faire passer du "JavaScript qui marche" au "JavaScript moderne qu'on écrit vraiment en production".

Ce que tu vas apprendre

Cette leçon est organisée en 6 chapitres progressifs. Chacun introduit une syntaxe ou un concept moderne, avec des exemples concrets et des exercices à tester dans la console.

Comment aborder cette leçon ?

Suis les chapitres dans l'ordre. Chaque notion s'appuie sur la précédente : les fonctions fléchées sont utilisées partout dans la suite, les objets littéraux préparent le terrain pour la déstructuration et les méthodes d'objets, et tout se combine dans les arrays d'objets — la structure de données la plus courante du web (listes de produits, d'utilisateurs, de résultats d'API...).

À chaque page, teste chaque exemple dans la console avant de passer au suivant. Comprendre en lisant c'est bien, mais c'est en tapant le code et en observant le résultat que ça rentre vraiment.

Prérequis

Avant de commencer, assure-toi d'avoir compris les notions de la Leçon 2 : boucles, conditions et surtout les fonctions (déclaration, paramètres, return). Le premier chapitre s'appuie directement dessus pour introduire la syntaxe fléchée.
